# Impaginare i blog post
1. giocare con blog_post_style.css per impaginare tutto automaticamente, l'idea è produrre un markdown file da emacs, metterlo come post e poi 11ty lo traduce in html
2. il file che c'è d'esempio è stato creato da emacs
3. rimane da occuparsi di tutti i livelli di indentazione dell'indice e poi del resto del post
4. Rimane anche da convertire tutti i <p>, questi tag vengono già usati nei project, bisogna tenere tutte quelle proprietà e fare find and replace per eliminarle
